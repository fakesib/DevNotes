# Linux

# SSH подключение

Для подключения к серверу по ssh, необходимо сначала сгенерировать ssh-ключ, приватный и публичный. 

**`ssh-keygen -t ed25519 -C "[your_email@example.com](mailto:your_email@example.com)"`**

Ключи генерируются по пути ~/.ssh

- `~/.ssh/id_ed25519` — **приватный ключ** (никому не передавать)
- `~/.ssh/id_ed25519.pub` — **публичный ключ** (его нужно скопировать на сервер)

Далее добавляем публичный ключ на сервер в файл authorized_keys:

**`echo "вставь_сюда_ключ" >> ~/.ssh/authorized_keys`**

И можно подключаться к серверу командой: 

`ssh user@0.0.0.0`
